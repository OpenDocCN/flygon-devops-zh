在本节中，您将通过实际操作学习如何以两种方式保护 Kubernetes 部署/集群：您将学习如何在构建、部署和运行时阶段保护 DevOps 流水线，并且您将了解深度防御，包括合规性、配置、身份验证、授权、资源管理、日志记录和监控、检测以及事件响应。

+   第六章，保护集群组件

+   第七章，认证、授权和准入控制

+   第八章，保护 Kubernetes Pod

+   第九章，DevOps 流水线中的图像扫描

+   第十章，实时监控和资源管理 Kubernetes 集群

+   第十一章，深度防御
