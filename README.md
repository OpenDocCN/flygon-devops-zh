## 下载

### Docker

```
docker pull apachecn0/freelearn-devops-zh
docker run -tid -p <port>:80 apachecn0/freelearn-devops-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g freelearn-devops-zh
freelearn-devops-zh <port>
# 访问 http://localhost:{port} 查看文档
```
